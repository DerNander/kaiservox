# EasyDictate Website

Landing page for [easydictate.app](https://easydictate.app)

## Deployment

### Option 1: GitHub Pages

1. Create a new repo (e.g., `easydictate-website`)
2. Push contents of this folder
3. Go to Settings → Pages → Deploy from branch `main`
4. Point your domain's DNS:
   - Type: CNAME
   - Name: @ (or www)
   - Value: YOUR_USERNAME.github.io

### Option 2: Vercel / Cloudflare Pages

1. Connect repo to Vercel/Cloudflare
2. Set build output directory to `/` (root)
3. Add custom domain in dashboard

## Files

- `index.html` - Main landing page
- `icon.svg` - App icon
- `favicon.svg` - Browser favicon
- `CNAME` - Custom domain for GitHub Pages

## Customization

Replace `YOUR_USERNAME` in index.html with your actual GitHub username.
Replace `XXXXX` in Microsoft Store link once published.
